#Short: Choyal Subscription Popup is plugin that can be use as newsletter subscription and support mailchimp.

#Brief: Choyal Subscription Popup is plugin that can be use as newsletter subscription and support mailchimp. 
Show/Hide popup and enable/disable mailchimp from plugin setting option. 
Subscribers data will be store in wordpress database.

#Features:
1. Subscription Popup 
2. MailChimp API Inetgration, 
3. Dashboard Subscriber's listing with delete function, bulk delete, Pagination, Search form
